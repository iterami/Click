iterami/Click.htm
-----------------

Click is a simple JavaScript-based idling game. Click the button, purchase upgrades, and see what highscores you can set! Progress is saved in localStorage.

---

Action           | Key
-----------------|----
Click            | C
Hide Settings    | -
Show Settings    | +
Upgrade Cluster  | 4
Upgrade Employee | 2
Upgrade Investor | 5
Upgrade Manual   | 0
Upgrade Script   | 1
Upgrade Server   | 3
